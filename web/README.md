# SPECA Web Notes

`web/` is not a runnable frontend application yet. It currently contains design
notes for a future read-only browser UI over SPECA outputs.

- `WEB_APP_DESIGN.md` records the current design proposal.
- There is no `package.json`, Vite app, Next.js app, or dev server in this
  directory yet.
- Current SPECA usage should start from the Codex App workflow described in the
  repository root `README.md` and `docs/CODEX_APP.md`.

When the web UI is implemented, replace this note with the actual install,
development, build, and deployment commands.
